# AWS-SaaS-Delivery-Copilot

Plataforma GenAI multi-tenant, orientada a consultoria AWS e clientes SaaS.

O app principal esta no diretorio `aws-saas-delivery-copilot/`.

O projeto esta na Etapa 5: fundacao FastAPI concluida, RAG local multi-tenant com fontes, Agent Toolkit deterministico, camada LLM Provider com mock local/Bedrock opcional e interface MCP-style para discovery e execucao de tools.

LangGraph, agentes autonomos, servidor MCP oficial com SDK, uso produtivo de Bedrock e deploy AWS ficam para etapas futuras.
